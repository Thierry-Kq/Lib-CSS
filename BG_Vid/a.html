Emulating background-size: cover for HTML5 video

It's a shame that there is no native way to make a video behave in the same way as a background with size set to “cover”, especially when we’re using video for backgrounds in responsive layouts.

There are plenty of plugins that will do this for you but it’s pretty easy using JavaScript (or jQuery in this example for laziness). Just make the video element keep it’s aspect ratio and scale to fit the parent container, hiding the overflow.

You can even set “focus” on a particular area of the video when it appears cropped using top and left percentages, just as you might for an image background using “cover” and “background-position”.
